# 專案變更紀錄

本文件記錄了對 `NTU_WEBAPP_three` 專案所做的前端變更。

## 檔案變更

- `index.html`: 新增了多個 HTML5 語意化標籤和常見網頁元素。
- `style.css`: 新增了 CSS 樣式來美化新增的元素及整體頁面排版。
- `script.js`: 新增了 JavaScript 檔案來處理互動功能。
- `README.md`: 建立此文件以說明變更內容。

## `index.html` 的主要變更

- **`<header>` 和 `<nav>`**: 新增了頁首和導覽列，提供網站的主要連結。
- **`<main>`**: 將頁面主要內容包裹起來，提升網頁結構的語意化。
- **`<section>` 和 `.card`**: 使用區塊來組織不同部分的內容（表格、圖片、表單），並以卡片樣式呈現。
- **`<img>`**: 新增了一個圖片元素，使用 placeholder 圖片作為範例。
- **`<form>`**: 加入了一個基本的聯絡表單，包含姓名、電子郵件輸入框和一個提交按鈕。
- **RWD (Responsive Web Design)**: 新增了 viewport meta 標籤，讓網頁在不同裝置上能正確縮放。
- **`<footer>`**: 新增了頁尾，包含版權資訊。
- **表格改進**: 將表格包裹在一個 `div.table-container` 中，以便在小螢幕上進行水平滾動。

## `style.css` 的主要變更

- **深色主題 (Dark Theme)**: 將網站整體風格從原本的淺色主題改為現代化的深色主題。
- **配色方案**:
    - 主要背景色改為深灰色 (`#121212`)。
    - 內容卡片、頁首和頁尾使用稍亮的灰色 (`#1E1E1E`)。
    - 主要文字顏色改為淺灰色 (`#E0E0E0`)，提升可讀性。
    - 互動元素（如按鈕和連結）的強調色改為藍色 (`#007BFF`)。
- **表單樣式**: 為輸入框 (`<input>`) 新增了符合深色主題的樣式。
- **互動效果**: 為導覽列連結增加了滑鼠懸停效果。
- **RWD (Responsive Web Design)**: 使用 CSS Media Query，當螢幕寬度小於 600px 時，導覽列會變為垂直排列，以適應手機等小螢幕裝置。
- **漢堡選單 (Hamburger Menu)**: 在手機版網頁上，將導覽列改為可點擊展開的漢堡選單，並透過 JavaScript 實現開關功能，提升行動裝置的使用體驗。
